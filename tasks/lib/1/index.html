题意

题目链接

大意给出 \(n,p,q,r\),\(n\) 个条件 \(a_i,b_i,c_i\)。

求有多少 \((a,b,c),(a\leq p,b\leq q,c\leq r)\),使得对于每个 \(i\),\(a>a_i\),\(b>b_i\),\(c>c_i\) 至少满足两个。

data range: \(n,p,q,r\leq 5\times 10^5\).

题解

考虑三维不好整，于是首先想到枚举其中一维，不妨设我们枚举 \(a\) 维。

设当前枚举的该维的值为 \(a\)。

那么这些条件可以分为两类:

  1. \(a>a_i\),那么 \(b,c\) 需满足 \(b>b_i \operatorname{or} c>c_i\).
  2. \(a\leq a_i\),那么 \(b,c\) 需满足 \(b>b_i \operatorname{and} c>c_i\).

发现如果把每一对 \((b,c)\) 看做平面上的点，

那么1可以看成一个L形区域。

2可以看成一个矩形区域。

满足 \(\forall i\) 条件的区域，就是每个满足的区域的交（显然）。

交不好维护，取补集，那么就是要维护并。

由于并的意义上增加比减少好做，如图，从绿色扩展到红色容易实现。

对于绿色正上方那些，直接区间覆盖。至于后面的，由于具有单调性，二分找出最后一个不大于 \(c_i\) 的，把不大于 \(c_i\) 的覆盖。

这里直接那线段树维护。

复杂度 \(\mathcal{O(n\log n)}\)

反思

要考虑好几何意义，注意单调栈维护轮廓线的使用。

我这个复杂度，常数均爆炸，十分不优秀。
